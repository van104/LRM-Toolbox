# Product Context

## 项目名称

LRM Tools (多功能在线工具箱)

## 核心价值

提供一个**轻量、无广告、纯前端运行（隐私安全）**的在线工具集合。

- **隐私保护**：所有处理逻辑均在浏览器端完成，敏感数据不上传服务器。
- **即开即用**：无需注册登录，无任何弹窗广告干扰。
- **全场景覆盖**：满足开发者、设计师及普通用户在日常工作中的高频需求。

## 核心功能

1. **多维度工具集**：
   - **开发工具**：JSON 格式化、Base64 编解码、JWT 解析、正则表达式测试、代码美化。
   - **媒体处理**：图片压缩/转换、二维码生成与扫描、PDF 拆分合并、SVG 预览。
   - **生活辅助**：单位换算、汇率转换、房贷计算器、随机数生成、日期差计算。
2. **离线能力 (PWA)**：支持渐进式 Web 应用，安装后可在无网络环境下正常使用。
3. **效率增强**：
   - **全局搜索**：支持拼音和关键词快速检索工具。
   - **收藏与历史**：支持工具收藏及本地历史记录记录，数据存储在 IndexedDB。
4. **高度自定义**：支持深色/浅色模式切换，工具面板布局可自由调整。

## 用户体验目标

- **极致性能**：首屏秒开，工具切换无缝连接。
- **响应式设计**：完美适配不同屏幕尺寸。
- **简洁直观**：遵循极简主义设计原则，降低用户认知成本。
